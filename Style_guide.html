Navigation Button

CSS selector:

.button-nav

Sample code:

<a class="button-nav" href="#">ABOUT THE COMPANY</a>

Rendered element:

ABOUT THE COMPANY

In-text link

CSS selector:

.in-text-link

Sample code:

<a class="in-text-link" href="#">SHOP MEN'S GLASSES</a>

Rendered element:

SHOP MEN'S GLASSES

Sample code:

<a class="button-signin" href="#">SIGN IN</a>

CANCEL SIGN IN

Rendered element:
